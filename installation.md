◆◆ 4DStreetViewMovieViewerの設定手順 ◆◆（全てターミナル上で行う。）  
＜準備編＞
1. Gitをインストールする  
   `git --version`  
     → バージョン情報が表示されると、既にインストールされている  
     → 入っていない場合、Xcodeをインストールする
2. Homebrewをインストールする
   `brew list`  
    → リストが表示されていれば既にインストールされている  
    　 下記コマンドを実行して、homebrewとgitの最新版のインストールを行う  
    　 `brew update`  
    　 `brew install git`
